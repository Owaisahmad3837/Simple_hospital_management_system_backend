# 🏥 Hospital Management System (Backend)

## 📌 Overview

This is a simple **Hospital Management System backend** built using **Python**.
It manages core hospital operations like doctors, patients, diseases, and appointments (enrollment system).

---

## 🚀 Features

* 👨‍⚕️ Add / Manage Doctors
* 🧑‍🦱 Add / Manage Patients
* 🩺 Store Patient Diseases
* 🧾 Doctor Specializations
* 📅 Appointment / Enrollment System
* 🔍 Basic record management using Python backend

---

## 🛠️ Tech Stack

* Python 🐍
* csv file
* OOP concepts

---



## ⚙️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/hospital-management-system.git
```

### 2. Go to project folder

```bash
cd hospital-management-system
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run project

```bash
python main.py
```

## 🧠 System Modules

### 👨‍⚕️ Doctor Module

### 🧑‍🦱 Patient Module
